Cas Concrets : L'impossible rendu possible

Quelques exemples où nous avons trouvé la solution quand tout semblait perdu

🏭

ERP SalesLogix abandonné

Système de 2000, code VBScript inmaintenable, 3 prestataires ont abandonné, développeur parti.

Notre solution : Reverse engineering complet, extraction des données, reconstruction des formulaires, migration progressive.
🏛️

Durcissement AD Secteur Public

Mairie avec obligations ANSSI, AD vulnérable, non-conformité réglementaire, risque cyber élevé.

Notre solution : HardenAD complet, conformité ANSSI/RGPD, migration 2022, audit trimestriel maintenu.
🌐

Intégration impossible

Système AS400 des années 80 à connecter avec API REST moderne, formats incompatibles.

Notre solution : Middleware PowerShell custom, transformation des données en temps réel, passerelle universelle entre systèmes hétérogènes.
💾

Ransomware - Tout chiffré

Serveurs chiffrés, backups corrompus, Veeam mal configuré depuis 2 ans.

Notre solution : Récupération depuis shadow copies oubliées, reconstruction AD depuis metadata, nouveau PRA testé trimestriellement.
📊

Performance SQL critique

Base SQL Server 2012, 500GB, temps de réponse 30s, DBA précédent a tout tenté.

Notre solution : Extended Events personnalisés, identification requêtes cachées, index manquants, temps réduit à 0.5s.
🏥

Hôpital - Sécurisation urgente

Infrastructure critique santé, exigences HDS, AD exposé, données patients à protéger.

Notre solution : Durcissement AD urgent, segmentation réseau, conformité HDS, monitoring 24/7 avec Zabbix.
